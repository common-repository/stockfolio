== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget if you want to use it. The widget's image will link to /stockfolio page
4. Insert [stockfolio] where you want to display your pie chart.
